# Secora

Enterprise cybersecurity infrastructure project developed as a diploma project.

## Overview

Secora is a cybersecurity-focused infrastructure project that simulates a real-world enterprise environment. The project includes Windows and Linux administration, centralized monitoring, threat detection, firewall management, SIEM integration, and security hardening.

## Key Features

* Active Directory Domain Services
* Group Policy Management
* IIS & Apache Web Servers
* Splunk SIEM Integration
* Wazuh XDR/SIEM Deployment
* Palo Alto NGFW Configuration
* Security Monitoring & Alerting
* Telegram & Email Notifications
* SSL Decryption & DLP
* Purple Team Exercises
* Infrastructure Hardening

## Technologies Used

* Windows Server
* Active Directory
* Linux (RHEL)
* Splunk Enterprise
* Wazuh
* Palo Alto NGFW
* Apache HTTP Server
* Docker/Podman
* PowerShell
* Bash
